     1→# SynapseLearn - Project Worklog
     2→
     3→## Project Overview
     4→AI-powered tutoring platform with slide upload, intelligent question generation, and an advanced multi-layered prompt system for adaptive learning. Built with Next.js 16, GSAP, Three.js, WebGL, Lenis smooth scrolling, and shadcn/ui.
     5→
     6→## Tech Stack
     7→- Next.js 16, TypeScript, Tailwind CSS 4, shadcn/ui
     8→- GSAP, Three.js/WebGL (@react-three/fiber + @react-three/drei), Lenis smooth scroll
     9→- Prisma ORM (SQLite), z-ai-web-dev-sdk for AI (LLM + TTS)
    10→- Zustand for state, TanStack Query for server state
    11→- Emerald/teal color theme (OKLCH), next-themes for dark mode
